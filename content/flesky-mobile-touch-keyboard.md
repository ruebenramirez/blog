Title: the fleksy touch type keyboard
Date: 09-01-2014 18:00
Category: review 

I came across an ad for flesky's cross platform touch type keyboard interface today and it inspired me to write about my experiences with mobile keybaords; both the touch type and physical variants.

This is the first keyboard that seems to be available across mobile platforms with both and Android and iOS installation.  (3rd party keyboards are a new feature in iOS8)  I installed flesky on my nexus 5 and it was pretty nice.  I don't know how well it will compete with iOS8's new keyboard and its conversation aware autocomplete feature, but it was a really nice alternative to the native android keyboard.

It all comes down to context of what you're typing.  If you're typing in a social context, like a tweet, a facebook post, or a txt message, then you're going to be really happy with the experience that flesky has worked so hard on.  And I totally get that this is the majority of mobile device users.  

If you're like me though, where most of what you type (on any of your devices) is more technical in nature, you're probably not going to be happy trying to find all the symbol keys.  They're not completely inaccessible, but they are definitely buried in the keyboard interface.  


I think google's/android's native keyboard is good enough on my nexus 5.  I don't have an iPhone, but I rarely use the touch keyboard on my iPad.  (I'm a pretty heavy user of Logitech's [Ultrathin keyboard cover](http://www.logitech.com/en-us/product/ultrathin-keyboard-cover))  When I do use the touch keyboard on my iPad it's typically in split mode.  If I'm going to be away from a power plug all day, I usually grab my iPad and [Incase's Origami Workstation](http://goincase.com/shop/incase-origami-workstation-for-ipad-2).
